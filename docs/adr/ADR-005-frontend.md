## Context

Платформа є закритою multi-tenant SPA (school management system) без вимог до SEO. Основні потреби:

- Динамічний UI
- In-memory зберігання `accessToken` для зменшення ризику XSS-крадіжки (ADR-003)
- Автоматичний silent refresh через `POST /auth/refresh` (refresh token у HttpOnly cookie)
- Захист маршрутів на рівні компонентів (`ProtectedRoute`)
- Команда має наявну експертизу в React-екосистемі

Серверний рендеринг (SSR) та SEO не є вимогами: платформа доступна лише авторизованим користувачам і не індексується пошуковими системами.

## Consequences

### Positive

- **Наявна експертиза** — команда знає React, що зменшує ризик затримок через навчання та дозволяє одразу використовувати перевірені патерни (React Query, Zustand, react-router).

### Negative

- **Відсутність SSR і React Server Components** — SPA є повністю клієнтським: весь JS bundle завантажується браузером, RSC недоступні (React Server Components вимагають серверного runtime — Next.js/Remix/тощо, якого у статичному SPA немає).

## Alternatives Considered

### Next.js (SSR/SSG)

**Розглядалось:** React-фреймворк із вбудованим SSR, file-based routing, API routes, оптимізацією зображень.

**Відхилено:**

- **SSR не дає цінності для закритої платформи.** Всі маршрути захищені (`ProtectedRoute`); сторінки не індексуються. Головна перевага SSR — SEO і TTFB для публічного контенту — тут нерелевантна.
- **Ускладнений деплой.** Next.js потребує Node.js-сервера або Vercel/cloud-специфічного хостингу.
- **Надлишкова функціональність без використання.** API routes, middleware, Image Optimization, ISR — всі ці можливості Next.js залишаться невикористаними, збільшуючи складність без віддачі.

### Vue.js (Vite SPA)

**Розглядалось:** Легший фреймворк із реактивністю з коробки, Pinia як state manager.

**Відхилено:** Команда не має Vue-експертизи. Екосистема менша ніж у React — більше часу на пошук рішень для специфічних задач . Ризик затримок через навчання неприйнятний в умовах часових обмежень.

### Angular

**Розглядалось:** Повноцінний фреймворк.

**Відхилено:** Значно більший бойлерплейт і крутіша крива навчання. Відсутня існуюча експертиза в команді.
